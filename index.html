Fast Browser

What is Fast Browser? Fast Browser is like any other browser. Made for surfing the web. This browser is probably the most privacy protecting browser - it doesn't store any data.

Technical details

Fast Browser is made in python using PyQt5. The installer and uninstaller are also written in python.

Libraries used in the installer/uninstaller:

  • PyQt5
  • Winreg
  • Pywin32
  • Urllib
  • Shutil
  • Threading
  • Os
  • Sys

    • Libraries used in the browser:

  • PyQt5
  • Keyboard
  • PyAutoGUI
  • Os
  • Sys
  • JSON
