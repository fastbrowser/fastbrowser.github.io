Fast Browser

(Note: Fast Browser is still in development. Some features may not work properly, however updates are to come.)

What is Fast Browser?

Fast Browser's purpose is the same as any other browser's. Made for surfing the web. This browser is the most privacy protecting browser - it doesn't store any data.

You can also view the source code and/or contribute by submitting pull requests here.

Technical details

Fast Browser is made in python using PyQt5. The installer and uninstaller are also written in python.

Libraries used in the installer/uninstaller:

  • PyQt5
  • Winreg
  • Pywin32
  • Urllib
  • Shutil
  • Threading
  • Os
  • Sys

    • Libraries used in the browser:

  • PyQt5
  • Keyboard
  • PyAutoGUI
  • Os
  • Sys
  • JSON
